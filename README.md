Future Daily Report Analyzer
============================

简介
----
    期货日报分析工具(下称'fdra')是一个工具，用于分析 [中国期货市场监控中心](https://investorservice.cfmmc.com) 给出的日报。  

安装依赖
-------

###  Fedora:
   sudo yum install python-xlrd sqlite

###  Windows:
   to be done.

用法概要
-------
>fdra import <日报文件>     解析导入逐日盯市日报文件
>
>fdra stat                  显示统计
>
